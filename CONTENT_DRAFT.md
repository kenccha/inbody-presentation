### [S1] 인터랙티브 퀴즈

**질문:** "체지방률을 가장 정확하게 측정하는 방법은?"

**선택지 (2x2 그리드):**
1. 🏋️ 체중계
2. 📏 BMI 계산
3. 🏊 수중체중법
4. ⚡ 생체전기 임피던스 (BIA)

**인터랙션:**
- 클릭 → 시뮬레이션 투표 바 (8% / 15% / 22% / 55%)

### [S2] Before InBody — 수중체중법

**2컬럼: SVG 일러스트 + 텍스트**

**SVG 요소:**
- 물탱크 (직사각형, 상단 열림, #005b9e 테두리)
- 물 (하단 70-75%, 반투명 블루, 수면 파도 애니메이션)
- 매달린 의자 (줄 + 의자, 물속)
- 잠긴 사람 (스틱 피규어, 반투명)
- 거품 (입 근처 → 위로 올라가는 애니메이션)
- 라벨: "HYDROSTATIC WEIGHING", "숨을 다 내뱉고 잠수", "×3회 반복"

**텍스트:**
- 라벨: "Before InBody — 01"
- 제목: "수중체중법"
- 본문: "물탱크에 완전히 잠긴 채 폐의 공기를 최대한 내뱉고 수중 체중을 측정합니다. 아르키메데스 원리로 체밀도를 구하고 체지방률을 계산하는데, 이 과정을 3회 반복해야 합니다. 연구실 전용 장비로 일반인 측정은 현실적으로 불가능했습니다."
- 태그: "물속 완전 잠수" / "노인·어린이 불가" / "오차 ±2.5%"

**리서치 검증 결과:**
- [v] 오차 범위: SEE ±1.5~2.8% — 확인됨
- [v] Siri/Brozek 공식으로 체밀도→체지방률 환산 — 확인됨
- [v] 잔기량(residual volume) 추정 시 3~4% 추가 오차 가능
- [v] 현재는 DXA가 골드 스탠더드로 대체됨
- [v] 지방무질량 밀도 1.1g/cm3 가정 → 노인(골다공증), 운동선수, 인종별로 부정확

### [S3] Before InBody — 캘리퍼 + RJL BIA

**2컬럼 카드**

**카드1: 피부두겹법 (Skinfold Caliper)**
- SVG: 피부층 + 캘리퍼 집게 + 양방향 화살표
- 제목: "피부두겹법 (Skinfold)"
- 본문: "캘리퍼로 피부를 집어 지방층 두께 측정. 측정자 숙련도에 크게 의존."
- 오차: "±3~9%"

**카드2: 누운 BIA — RJL 방식**
- SVG: 누운 스틱 피규어 + 4전극(손목2, 발목2) + 점선 연결
- 제목: "누운 자세 BIA (RJL 방식)"
- 본문: "누워서 손목·발목에 전극 4개 부착. 50kHz 단일 주파수만 사용. 체액이 자세에 따라 이동해 10분 안에도 값이 바뀜."
- 태그: "자세 의존 · estimation 기반"

**리서치 검증 결과:**
- [v] RJL Systems: 1979년 최초 상용 임피던스 미터 출시. 50kHz 단일주파수. 전신을 1개 원통 모델.
- [v] Lukaski, Segal 등 1980년대 한계 발견: 단일주파수로 ICW/ECW 구분 불가
- [v] RJL 총 예측 오차: 4.4kg (비교 연구). 탈수만으로 FFM 5kg 과소추정 가능
- [v] Skinfold: 3~7부위 측정, Jackson-Pollock/Durnin-Womersley 등 다수 공식 존재

### [S5] InBody의 3가지 혁신

**3컬럼 카드:**

**카드1: Multi-Frequency (다주파수)**
- 🔀 아이콘
- "1kHz~3MHz, 최대 8개 주파수. 세포 내·외 수분을 정밀 구분. 세계 최초 3MHz 고주파 적용 (2020)."
- 서브: "DXA 대비 98% 상관관계 (Mayo Clinic 검증)"

**카드2: 8점 접촉 전극**
- ✋🦶 아이콘
- "양손 엄지+손바닥, 양발 앞꿈치+뒤꿈치. 5개 부위 독립 측정. 재현성 99%."
- 서브: "1996년 차기철 박사가 세계 최초 개발"

**카드3: No Estimation**
- 🚫📊 아이콘
- "나이·성별·인종 등 통계 추정값 배제. 오직 측정된 임피던스만으로 결과 도출."
- 서브: "8,000편+ 학술 논문에서 reference device로 사용"

**리서치 검증 결과:**
- [v] InBody 970: 8개 주파수 (1, 5, 50, 250, 500kHz, 1, 2, 3MHz) = 40개 임피던스 측정
- [v] InBody 970S (현행): 6개 주파수 (5, 50, 250, 500kHz, 1, 3MHz) = 30개 측정
- [v] 3MHz 세계 최초: 2020년 InBody 970에서 도입. 반박 자료 없음.
- [v] 논문 수: InBody USA 기준 8,000편+ (브리프의 5,000은 구 수치)
- [v] 8점 전극: 1996년 차기철(Dr. Kichul Cha) 개발
- [v] No Estimation: 나이/성별 입력은 받지만 임피던스 계산에는 사용하지 않음
- [v] DXA 상관관계 98% (Mayo Clinic 2020 검증 연구)

### [S6] 세포막 다이어그램

**2컬럼: SVG 세포 다이어그램 + 텍스트**

**SVG 요소:**
- 큰 원 = 세포 (테두리 = 세포막, 브랜드 색상)
- 상단 라벨: "CELL MEMBRANE — 인지질 이중층 = 캐패시터"
- 원 안 점선 원 = ICW
- 원 밖 = ECW
- 왼쪽 빨간 화살표 → 세포막에서 ✕ = "저주파 — 세포막 통과 불가"
- 오른쪽 브랜드 색상 점선 화살표 → 관통 = "고주파 — 세포막 관통"
- 점선 이동 애니메이션 (stroke-dashoffset)

**텍스트:**
- 라벨: "여러분의 전공이 이 기계 안에 있다"
- 제목: "세포막이 캐패시터가 된다"
- 본문: "인지질 이중층이 축전기처럼 작동합니다..."
- 강조 박스: Phase Angle = 세포막 무결성 지표. 암환자 예후, 투석환자 영양상태, 사코페니아 진단.

**리서치 검증 결과:**
- [v] Phase Angle = arctan(Xc/R) × (180/π) — 확인됨 (보통 50kHz에서 측정)
- [v] 정상 범위: 건강 성인 5~7도 (일부 출처 4~10도)
- [v] 임상 활용 확인:
  - 암환자: 낮은 PhA → 불량한 예후, 수술 후 합병증 위험 증가
  - 투석환자: PhA로 영양상태 모니터링
  - 사코페니아: PhA 1도 감소 시 사코페니아 위험 유의미 증가. AWGS 2025 합의문에서 다주파수 BIA 인정
  - 심장수술 전: PhA로 영양 위험도 평가 → ICU 재원일수/병원 사망률 연관
- [v] 등가회로 모델(Cole model): Re(세포외 저항) // [Cm(세포막 캐패시턴스) + Ri(세포내 저항)]
- [!] 주의: 2023 Reviews in Endocrine & Metabolic Disorders 리뷰 — 많은 PhA-예후 연관이 경험적/관찰적
